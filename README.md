
# Credit Card Analytics Dashboard

This project presents an interactive dashboard to analyze credit card customer behavior, transaction patterns, and revenue performance. The report provides key insights to help understand customer spending habits and overall business performance.

---

## Overview

The objective of this project is to analyze credit card data and visualize important metrics such as revenue, transaction volume, customer segments, and spending categories. The dashboard helps identify high-value customers and key drivers of revenue.

---

## Key Insights

* Analyzed total revenue, transaction count, and average spending
* Identified high-value customer segments based on demographics
* Evaluated spending patterns across different categories
* Performed state-wise and segment-wise performance analysis
* Highlighted key trends to support business decision-making

---

## Dashboard Features

* Interactive filters and slicers
* KPI cards for revenue and transactions
* Category-wise spending analysis
* Customer demographic insights
* Geographic performance visualization

---

## Tools Used

* Power BI
* Power Query (Data Cleaning & Transformation)
* DAX (Calculated Measures & KPIs)
