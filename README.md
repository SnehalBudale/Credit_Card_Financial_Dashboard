# Credit Card Financial Dashboard
Credit Card Financial Dashboard is a Power BI-based project that builds an interactive dashboard for visualizing and analyzing credit card transactions and customer data. 

Collected and cleaned credit card transaction and customer data using SQL.

Designed an interactive Power BI dashboard to analyze spending patterns, revenue, and customer behavior.

Created KPIs and visualizations for transaction trends, card usage, and financial performance.

Enabled data-driven insights through filters, slicers, and dynamic reports.
